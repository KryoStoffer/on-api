Övergripande

Övergripande Information

Endpoints

En URL pekar ut adressen till roten API:et, ex. http://host/api/2.3.1/.

Under URL:en förväntas tjänsterna finnas: orders/ (http://host/api/2.3.1/orders/), accesses/ (http://host/api/2.3.1/accesses/) etc.

Autentisering

Autentisering sker med HTTP BASIC. Se RFC-2617.

Character Encoding

Samtliga API:er använder UTF-8.
